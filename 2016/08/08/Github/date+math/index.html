Date + Math

3.Date

Date.getDate()

返回一个Date对象的月份中的日期值 1~31

Date.getDay()

返回一个Date对象的一周中的日期值 0~6

Date.getFullYear()

获取本地时间的年份，如：2017

Date.getHours()

时

Date.getMinutes()

分钟数。

Date.getSeconds()

秒

Date.getMilliseconds()

本地时间表示时的毫秒值

Date.getTime() 方法返回一个时间的格林威治时间数值。(时间戳)

4.Math

Math.PI 表示一个圆的周长与直径的比例，约为 3.14159：

function calculateCircumference (radius) {
  return 2 * Math.PI * radius;
}

calculateCircumference(1);  // 6.283185307179586

Math.random()

返回0到1之间的伪随机数.

Math.abs(x)

返回x的绝对值.

Math.cbrt(x)

返回x的立方根.

Math.ceil(x)

返回x向上取整后的值.

Math.floor(x)

返回小于x的最大整数。

Math.pow(x,y)

返回x的y次幂.

Math.pow(3,3)//27
